# Password Program Project
---
Password program project is a sample console application that implements the requirements
outlined in [Company password policy](PasswordProgramPolicy_Clark.md). 
best practices for secure password management and authentication mechanisms. 

The project includes features such as password hashing, salting, and 
multi-factor authentication (MFA) to ensure robust security for user credentials.

---

## Features

- Password complexity enforcement
- Password expiration and history management
- Account lockout after multiple failed login attempts

---
## Implementation
The project is implemented in C# and targets .NET 10.0 framework.
